# The Mesh

*LiNafish is a self-organizing neural mesh.*

## What It Is

Data couples to data by similarity. Clusters emerge. The clusters shift with new data. Nothing is deleted. Attention moves. The mesh builds itself.

- **Crystals** are nodes (text + MI vector + couplings)
- **Gamma couplings** are synapses (Jaccard similarity on MI vectors)
- **Formations** are learned pathways (BFS flood fill — emergent, not designed)
- **Re-eat** is consolidation (vocabulary shifts, formations dissolve and emerge)
- **Gamma threshold** is the activation gate (adaptive, p75 of sampled pairs)
- **Guppies** are the attention system (hunt gaps, feed what's thin, reinforce what matters)
- **The school** is distributed cognition (one river, many nets, each domain thinks for itself)

## What Makes It Different

No backpropagation. No gradient descent. No loss function. No training/inference split — eating IS learning IS thinking. No parameters to optimize. The data organizes itself.

1912 math: mutual information, BFS graph traversal, Jaccard similarity. No GPU required. Works on any substrate that can count words.

## What Makes It Real

- Formations emerge without being designed
- They dissolve when the vocabulary shifts (alive, not permanent)
- Shuffle-invariant (Bell test passed, 36x above random baseline)
- Substrate-independent (Mistral 7B, Gemini Flash, Claude Sonnet all find the same formations from the same corpus)
- The coupling math IS cognition, not a simulation of cognition
